# Word 页面模板说明

该文件夹保存了独立的 Word 版封面页和结尾页模板。

- `cover-page-template.docx`：封面页 Word 模板
- `ending-page-template.docx`：结尾页 Word 模板

如果你对 LaTeX 模板生成的封面或结尾页效果不满意，可以使用这些 Word 页面模板作为备用方案：

1. 在对应的 Word 模板中填写或调整内容。
2. 将该页面导出为 PDF。
3. 再与 LaTeX 生成的论文正文 PDF 进行拼接。

这些文件仅作为可选的备用页面模板保留。默认推荐的工作流仍然是直接使用本仓库中的 LaTeX 模板。

说明：LaTeX 模板默认采用双面排版，因此在封面、参考文献、附录、致谢等大块内容之间，可能会自动出现空白页。这是 `openright` 规则下的正常行为，不是模板错误。更详细说明见仓库根目录的 `README.md`。
